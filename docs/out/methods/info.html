/getInfo ¶

Gets information about a user

URL Query Parameters

Key Value
token Any token to authorize this request
userid User ID of the user to get info of

Example response 

{
    "success": true,
    "result": {
        "user_id": 570787098,
        "banned": false,
        "reason": "",
        "message": "",
        "ban_source_url": "",
        "banned_by": 0,
        "crime_coefficient": 93,
        "date": "2021-11-15 at 14:50:24",
        "source_group": "",
        "ban_flags": null,
        "is_bot": false
    },
    "error": null
}

/checkToken ¶

Checks if a token is valid

URL Query Parameters

Key Value
token A token to check if valid

Example response with a valid token 

{
    "success": true,
    "result": true,
    "error": null
}

Example response with an invalid token 

{
    "success": true,
    "result": false,
    "error": null
}
