/report Â¶

Reports a user

URL Query Parameters

Key Value
token An enforcer, inspector or owner token to authorize this request
userid User ID of the user to report
reason The report reason
message Optional contents of a message accompanying the report
source Optional link (to a message?) accompanying the report
source-group Optional link to a group accompanying the report
bot Optional parameter to specify if the user being reported is a bot

Example response 

{
    "success": true,
    "result": "User has been reported",
    "error": null
}
